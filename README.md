# Eye Disease Classification with Chatbot Integration

This project aims to develop an AI-powered chatbot system for diagnosing eye diseases using user-provided clinical data (age, blood pressure, sugar level, gender) and retinal scan images. The chatbot also provides actionable recommendations and guides users to relevant healthcare services.

## Key Features
- **AI Disease Classification**: Convolutional Neural Networks (CNN) for retinal image analysis and machine learning models for processing clinical data.
- **Chatbot Interface**: A user-friendly chatbot that interacts with users, collects necessary data, and delivers diagnoses or health tips.
- **Multi-Modal Integration**: Combines clinical (text) and visual (image) inputs for improved diagnostic accuracy.
- **Additional Services**: Provides users with preventive care advice, doctor recommendations, and eye health updates.
